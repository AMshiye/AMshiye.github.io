Hexo 使用指南（小白版）

本文档是给初学者的 Hexo 使用指南，覆盖最常用的操作流程与命令。

1. 基本概念

  • Hexo 是静态博客生成器。
  • 你写 Markdown，Hexo 生成静态网页。
  • 本地预览无风险，部署后才会上线。

2. 常用目录

  • source/_posts/：文章目录（写作主要位置）
  • source/about/：About 页面
  • scaffolds/：新建文章模板
  • _config.yml：站点配置
  • themes/cactus/：主题目录

你必须先完成 SSH 配置并在 _config.yml 中配置 deploy。

7. 修改站点信息

编辑 _config.yml：

  • title：网站标题
  • author：作者名
  • description：网站描述
  • language：语言（如 zh-CN）

9. 常见问题

9.1 文章不显示

  • 检查文件是否在 source/_posts/
  • 检查文章前置的 YAML 是否正确

9.2 预览不更新

  • 先执行 npx hexo clean
  • 再执行 npx hexo server

9.3 部署失败

  • 多数是 SSH 未配置或仓库地址错误
  • 请参考 DEPLOY.md
